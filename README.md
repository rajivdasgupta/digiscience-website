# Digiscience Techsol Premium Website

## Run locally
```bash
npm install
npm run dev
```

## Push to GitHub
```bash
git init
git add .
git commit -m "Initial premium website"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/digiscience-techsol-website.git
git push -u origin main
```

## Deploy to Vercel
1. Open Vercel
2. Add New Project
3. Import the GitHub repo
4. Deploy

## Domain
You can then connect:
digisciencetechsol.com
